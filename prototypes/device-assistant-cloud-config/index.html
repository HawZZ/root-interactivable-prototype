编辑新手指导

W1 / 新版设备助手 / 英文（默认） / 草稿与发布

APP 预览

左侧为所见即所得预览；点击步骤同步定位右侧配置。

页面与步骤

选择 Welcome / Step / Finish / 说明书 PDF，再编辑对应模板。
Step 模板可创建 1～7 页；顺序即 APP step 页顺序。

页面配置

编辑当前语言的固定模板字段。
保存草稿仅暂存于平台，APP 不可获取。

导入语言配置

选择来源语言后，覆盖当前语言配置并保存为草稿。
优先导入来源语言的已发布配置；若来源语言未发布但存在草稿，则导入草稿配置。
