①
触发条件区域
核心优化
改动范围

原页面「触发条件」为单一 Select 下拉，V0.3 优化后根据触发类型展开对应配置面板。设备触发、云端触发、耗材触发均为本期验收范围。

触发类型定义
  • 设备触发：选择物模型属性，设置判断符和属性值触发
  • 云端触发：选择云计时器，设置剩余时长的条件值触发；“剩余时长”语义固定为 ≤
  • 耗材触发：选择耗材管理项，选择「耗材不足」或「耗材耗尽」事件触发
切换行为
  • 从任一触发类型切换到其他类型时，静默清理已选项
  • 已选摘要区变为空态占位文字（不弹确认框）
✓ AC-007 覆盖此行为
②
触发条件 Select
还原真实样式

触发条件入口保持平台统一的 Select 下拉控件样式，与其他字段保持一致。

选中任一触发类型后，Select 下方展开对应配置面板（非替换，是追加展开）。

切换其他选项时收起当前面板并清理已选项。

③
属性列表三列
P0 必做
字段
  • 属性名称：中文名，溢出省略
  • 标识符：等宽字体，唯一键，区分同名属性
  • 数据类型：Tag 展示，颜色见批注④
选中交互
  • 单选，点击行高亮（蓝色左边框 + 浅蓝背景）
  • 点击另一行时旧高亮消失（不可多选）
  • 保存字段为单个 identifier 字符串
条件设置
  • 选择属性后设置判断符：= / ≠ / > / ≥ / < / ≤
  • 填写属性值，摘要区同步展示「属性 + 判断符 + 属性值」
  • 属性值输入控件类型可在正式设计中按 dataType 细化，本原型先用通用输入框验证链路
容器规格

列表区固定高度 228px，超出内容内部滚动，不撑开页面。

③-云
云端触发 - 云计时器列表
P0 必做
选择对象

选择产品关联的云计时器（如滤芯寿命计时、UV灯管计时等）。

列表字段
  • 云计时器名称：中文名
  • 标识符：等宽字体，唯一键
  • 状态：运行中 / 已暂停
条件配置
  • 条件值：剩余时长（分钟）
  • 不提供判断符选择；“剩余时长”的业务语义即 ≤
  • 示例：剩余时长 ≤ 30 分钟时触发
选中交互
  • 单选，点击行高亮
  • 选择后填写剩余时长条件值
③-耗
耗材触发 - 耗材管理项列表
P0 必做
选择对象

选择产品关联的耗材管理项（如主滤芯、前置滤芯、UV灯管等）。

列表字段
  • 耗材管理项名称：中文名
  • 标识符：等宽字体，唯一键
  • 类型：滤芯 / 灯管 / 液体 / 耗材
事件配置
  • 触发事件：耗材不足 / 耗材耗尽
  • 选择耗材管理项后需选择触发事件
选中交互
  • 单选，点击行高亮
  • 选择后需选择触发事件

产品上下文异常状态
P1-003
触发条件

产品字段在进入创建页时已确定，创建页内只读展示，不提供编辑入口。仅当入口产品上下文异常或对象数据无法加载时，触发条件区域不可配置并显示提示。

UI 表现
  • 显示提示文字：「产品上下文异常，无法加载产品关联的触发对象，请返回产品详情页重新进入」
  • 触发对象选择器不展示，避免在无产品上下文时误选

点击顶部「产品上下文异常」场景切换按钮演示。
